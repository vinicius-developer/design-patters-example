## GoodCode

Nesse caso nós iremos unir as duas entidades em uma 
Fabric e ao invés dela ser uma Fabric normal passará a 
ser uma Abstract Factory já que instância vários tipos 
de classe que representam algo maior.

Uma grande vantagem desse padrão é que ele permite criar 
agrupamentos, ou seja é quase impossível nós criarmos 
o imposto errado para nossaa Venda. 

E também essa forma dimunui uma o acoplamento do código
em nosso arquivo de mais alto nível
