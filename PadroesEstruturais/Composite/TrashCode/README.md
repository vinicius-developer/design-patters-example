##TrashCode

Nesse exemplo não temos um código necessáriamente ruim. 

Nós possuimos duas classes **Item** e **Orcamento**. A classe
Item só possue um atributo chamado valor e esse e a classe Orcamento
só possui um atributo chamado itens que é um array com vários items.

Nossa classe Orcamento possui dois método um para adicionar um item
no Orcamento e outro para cacular o valor total do orcamento. A grande 
questão aqui e se for necessário criar o valor de um orcamento a partir 
de um orcamento mais antigo? Essa é a pegunta que vai nos fazer implementar
o padrão Composite
