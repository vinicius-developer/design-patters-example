## GoodCode

Agora em nossa classe de Orcamentos podemos adiconar um Orcamento 
também como item já que ele implementa a interface CompositeOrcamento
que faz com que obriatóriamento ele também tenha um método **valor**
assim como a classe **Item**.

Dessa forma quando chamamos o método valor não importa se é um Item ou 
Orcamento ele será somado igualmente no valor final.
