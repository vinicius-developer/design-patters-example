## GoodTrash 

Nesse trecho de código nós criamos uma classe proxy que armazena 
informações do método **verifyPermission** em um atributo e caso esse
atributo já tenha algum valor ele retona essa valor caso não ele chama 
o método e salva no atributo

Então com isso nós só precisamos ter o tempo de espera uma vez o resto 
estará salvo na memória do programa
