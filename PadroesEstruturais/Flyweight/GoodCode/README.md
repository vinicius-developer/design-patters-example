## GoodCode

Quando nós aplicamos esse padrão de projeto básicamente 
extraimos os items da classe que podem ser externos para 
que ele não precisem ser craidos em todoas as iterações 

Esse dados que são extraidos precisam ser iguais em todos os 
objetos e imutaveis também 

Tome cuidado com esse padrão de projeto, pois muitas vezes acabamos
criado classes externas com atributos que não tem muita semelança
